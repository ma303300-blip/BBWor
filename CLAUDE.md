# כללי עבודה לפרויקט BBWor

## סדר עבודה חובה לכל שינוי

1. בצע את התיקון / התוספת
2. עשה `commit` ו-`push`
3. **בדוק שהכל עובד** — עם Playwright, בדיקת לוגיקה, או כל שיטה מתאימה
4. רק אחרי אישור שהכל תקין — צור PR ומזג
5. דווח למשתמש מה בדקת ומה התוצאה

**אסור למזג בלי לבדוק קודם.**

## ניקוי אחרי מיזוג

אחרי כל מיזוג מוצלח:

1. עבור ל-main ועדכן:
   ```
   git checkout main && git pull origin main
   ```
2. מחק את הענף המקומי:
   ```
   git branch -d <branch-name>
   ```
3. נקה ענפים ישנים שכבר מוזגו:
   ```
   git fetch --prune
   ```
4. נקה קאש git מיותר:
   ```
   git gc --auto
   ```
